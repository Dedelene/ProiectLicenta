~~~~Escape Room - Joc educational~~~

Joc 3D de tip escape room cu puzzle-uri economice realizat in Unity.
Scopul jocului este insusirea/atribuirea unor cunostinte economice in vederea rezolvarii puzzle-urilor si a evadarii din casa unui geniu economist.

~~~~
Descriere
~~~~

Motor grafic: Unity
Gameplay: first-person
Limbaj: C#
